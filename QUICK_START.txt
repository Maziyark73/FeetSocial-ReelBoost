1️⃣  INSTALL DEVELOPMENT TOOLS
   
   Install Xcode Command Line Tools:
   → Run: xcode-select --install
   → Wait for installation dialog and accept

   Install Homebrew (if needed):
   → /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   
   Install Node.js:
   → brew install node
   
   Install PM2 globally:
   → npm install -g pm2

2️⃣  INSTALL DEPENDENCIES
   
   cd reel-boost-backend && npm install && cd ..
   cd reel-boost-website && npm install && cd ..
   cd reel-boost-admin && npm install && cd ..

3️⃣  CONFIGURE ENVIRONMENT VARIABLES
   
   Backend:
   → cd reel-boost-backend
   → cp .env.example .env
   → Edit .env with your database credentials
   
   Website:
   → cd reel-boost-website
   → cp .env.example .env.local
   → Edit .env.local with API URLs and keys
   
   Admin:
   → cd reel-boost-admin
   → cp .env.example .env
   → Edit .env with API URL

4️⃣  INITIALIZE GIT & PUSH TO GITHUB
   
   git init
   git add .
   git commit -m "Initial commit: ReelBoost monorepo setup"
   git branch -M main
   
   # Create "FeetSocial-ReelBoost" as a private repo on GitHub first
   git remote add origin git@github.com:YOUR_USERNAME/FeetSocial-ReelBoost.git
   git push -u origin main

🔑 ENVIRONMENT VARIABLES TO CONFIGURE:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

BACKEND (reel-boost-backend/.env):
  • DB_USERNAME - MySQL username
  • DB_PASSWORD - MySQL password  
  • DB_DATABASE - MySQL database name
  • DB_HOST - MySQL host (127.0.0.1)
  • Port - Server port (3000)
  • screteKey - JWT secret (generate strong random string)
  • baseUrl - Server base URL
  • MEDIAFLOW - Storage type (LOCAL/S3)

WEBSITE (reel-boost-website/.env.local):
  • NEXT_PUBLIC_API_URL - Backend API URL
  • NEXT_PUBLIC_SOCKET_URL - Socket.io URL
  • NEXT_PUBLIC_GOOGLE_SIGNIN_ID - Google OAuth Client ID
  • NEXT_PUBLIC_GOOGLE_MAP_KEY - Google Maps API Key
  • NEXT_PUBLIC_GIF_API_KEY - Giphy API Key

ADMIN (reel-boost-admin/.env):
  • VITE_API_URL - Backend API URL
